Glosario

Ir a Glosario (Ñ - Z)
A B C D E F G H I J K L M N

A

Algoritmo: Conjunto de pasos ordenados para resolver un problema específico.

B

Bug: Error o fallo en un programa que produce un resultado no esperado.

C

Clase: Plantilla en programación orientada a objetos para crear objetos con atributos y métodos.

D

Dominio: Nombre único que identifica un sitio web en Internet.

E

Encapsulamiento: Técnica que restringe el acceso directo a los datos de un objeto.

F

Framework: Conjunto de herramientas y componentes reutilizables para facilitar el desarrollo de software.

G

Gateway: Dispositivo que conecta redes diferentes y traduce protocolos.

H

Hash: Valor generado a partir de datos mediante una función matemática para identificar información rápidamente.

I

Índice: Estructura que mejora la velocidad de búsqueda en bases de datos.

J

JSON: Formato ligero de intercambio de datos fácil de leer y escribir para humanos y máquinas.

K

Keylogger: Programa que registra las pulsaciones del teclado de un usuario.

L

Lenguaje de programación: Sistema de comunicación para escribir instrucciones que una computadora puede ejecutar.

M

Middleware: Software que conecta diferentes aplicaciones o servicios dentro de un sistema.

N

Namespace: Espacio que organiza y agrupa identificadores únicos en programación.
