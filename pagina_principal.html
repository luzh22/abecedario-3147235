Glosario

Ir a Glosario (Ñ - Z)
A B C D E F G H I J K L M N

A

Algoritmo:

Conjunto de pasos ordenados para resolver un problema o realizar una tarea.

B

Bug:

Error o fallo en un programa que produce un resultado no esperado.

C

Compilador:

Programa que traduce código fuente a código máquina o a otro lenguaje.

D

Depuración:

Proceso de identificar y corregir errores en el código.

E

Estructura de datos:

Forma de organizar y almacenar datos para facilitar su uso y acceso.

F

Función:

Bloque de código reutilizable que realiza una tarea específica.

G

Git:

Sistema de control de versiones distribuido para gestionar el código fuente.

H

HTML:

Lenguaje de marcado utilizado para estructurar páginas web.

I

Intérprete:

Programa que ejecuta instrucciones de código fuente línea por línea.

J

JavaScript:

Lenguaje de programación utilizado principalmente para desarrollo web.

K

Kernel:

Núcleo del sistema operativo que gestiona los recursos del sistema.

L

Librería:

Colección de funciones y recursos reutilizables en programación.

M

Método:

Función asociada a un objeto o clase en programación orientada a objetos.

N

Nodo:

Elemento individual de una estructura de datos como listas o árboles.
