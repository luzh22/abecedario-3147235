Glosario (Ñ - Z)

Ir a Glosario (A - N)
Ñ O P Q R S T U V W X Y Z

Ñ

Ñandú: Nombre de una función ficticia utilizada para ilustrar ejemplos en documentación técnica.

O

Objeto: Entidad que agrupa datos y métodos en la programación orientada a objetos.

P

Parámetro: Valor que se pasa a una función o procedimiento para personalizar su comportamiento.

Q

Queue: Estructura de datos que sigue el principio FIFO (primero en entrar, primero en salir).

R

Repositorio: Lugar donde se almacenan y gestionan archivos de código y documentación de un proyecto.

S

Servidor: Computadora o programa que proporciona servicios o recursos a otros dispositivos en una red.

T

Thread: Unidad de ejecución dentro de un proceso que permite realizar tareas simultáneas.

U

Unicode: Estándar de codificación que permite representar caracteres de la mayoría de los sistemas de escritura.

V

Vector: Arreglo unidimensional que almacena una colección de elementos del mismo tipo.

W

Widget: Elemento gráfico de una interfaz de usuario, como un botón o una barra de progreso.

X

XSS:

Tipo de vulnerabilidad de seguridad en aplicaciones web que permite la inyección de scripts maliciosos.

Y

Yield:

Palabra clave utilizada en algunos lenguajes para pausar y reanudar funciones generadoras.

Z

Zona horaria:

Región geográfica que comparte la misma hora estándar para la organización del tiempo.
