Glosario (Ñ - Z)

Ir a Glosario (A - N)
Ñ O P Q R S T U V W X Y Z

Ñ

Ñu:

Nombre coloquial para GNU, un sistema operativo libre compuesto enteramente de software libre.

O

Open Source:

Software cuyo código fuente es accesible y puede ser modificado y distribuido libremente.

P

Pseudocódigo:

Descripción de un algoritmo utilizando una notación similar a un lenguaje de programación, pero más informal.

Q

Query:

Consulta realizada a una base de datos para obtener información específica.

R

Recursividad:

Técnica en la que una función se llama a sí misma para resolver un problema.

S

Script:

Archivo que contiene un conjunto de instrucciones que son ejecutadas por un programa o intérprete.

T

Token:

Unidad léxica mínima en la programación, como una palabra clave, identificador u operador.

U

UI (Interfaz de Usuario):

Parte de una aplicación con la que el usuario interactúa directamente.

V

Variable:

Espacio en memoria que almacena un valor que puede cambiar durante la ejecución de un programa.

W

Web:

Sistema de documentos interconectados y accesibles a través de Internet mediante navegadores.

X

XML:

Lenguaje de marcado utilizado para almacenar y transportar datos de forma estructurada.

Y

YAML:

Lenguaje de serialización de datos legible por humanos, usado para archivos de configuración.

Z

ZIP:

Formato de compresión de archivos utilizado para agrupar y reducir el tamaño de archivos y carpetas.
